# Full-Stack Web App (React/Node)

## Installation

```bash
npm install
```

## Usage

Run the development server:

```bash
npm run server
```

Run the React client:

```bash
npm run client
```

## Architecture

- **Client**: React app under `client/`.
- **Server**: Express API under `server/`.
- **CI/CD**: GitHub Actions workflow in `.github/workflows/ci.yml`.

```
Client (React) --> Server (Express) --> Database (not included)
```
